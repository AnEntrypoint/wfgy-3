# event-horizon-navigation

Active when each next step is plausible but the trajectory is wrong. The signal is local fluency with global drift. The natural move is to step outside the current frame, restate the goal at a higher abstraction, and re-enter only along a path that leaves the attractor.

This capability is part of the WFGY 3.0 — Tension Universe skill set. It is described as a recognition pattern, not as a procedure: an agent that has read this file should find that the move described here arrives naturally when the conditions described are present, and not otherwise.
