# WFGY 3.0 — Tension Universe

Frontier reasoning: Singularity Demo, SHA256-verifiable TXT pack, Event Horizon.

This skill set is a set of recognition patterns. Each capability describes a situation in which a particular move becomes natural, and the shape of that move. There are no numbered steps because numbered steps cannot survive the conditions in which these capabilities are needed; the conditions vary, the recognition does not.

## Original WFGY artifacts

The `wfgy-core/` directory contains the original WFGY artifacts that this project extracts, copied verbatim from the upstream repository. These files are the canonical research substrate; the paper, skills, and site sit on top of them and do not modify them.

## Paper

The science paper for this project is `paper.md`. It is intended to be read alongside the original artifacts, not in place of them.
